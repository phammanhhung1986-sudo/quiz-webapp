# Quiz Webapp - Full Feature (Vite + React)

## Giới thiệu
Đây là project demo full-featured theo yêu cầu:
- Upload Excel qua Supabase Storage (hoặc localStorage fallback)
- Parse Excel bằng SheetJS
- Quiz timer 30s, highlight đúng/sai, animations
- Results: pie chart + progress
- Admin page: upload, list, select, delete files
- Leaderboard lưu localStorage
